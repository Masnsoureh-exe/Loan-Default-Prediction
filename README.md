Loan Default Risk Prediction

پیش‌بینی احتمال عدم پرداخت متقاضی وام، با استفاده از اطلاعات اجتماعی، مالی و شغلی متقاضی.

Problem
با توجه به پروفایل متقاضی (درآمد، سن، شغل، محل سکونت، مالکیت خانه/خودرو، سابقه شغلی و...)، هدف پیش‌بینی متغیر دودویی Risk_Flag است؛ یعنی اینکه آیا احتمال دارد متقاضی در بازپرداخت وام نکول کند یا نه.

approach
- تحلیل اکتشافی داده (EDA)
  بررسی توزیع ویژگی‌ها، مقادیر گمشده، داده‌های تکراری، و رابطه هر ویژگی دسته‌ای با نرخ نکول (شهر، شغل، استان، مالکیت خودرو/خانه، وضعیت تأهل)
- بررسی داده‌های پرت
  استفاده از روش IQR روی ویژگی درآمد (هیچ داده پرتی یافت نشد)
- مهندسی ویژگی (Feature Engineering)
  - تبدیل ویژگی سن به یک ویژگی دسته‌ای به نام age_group (جوان / میانسال / مسن)
  - ساخت ویژگی job_stability_ratio (نسبت سابقه شغلی فعلی به کل سابقه کاری)
- Encoding
  - رمزگذاری دودویی برای ویژگی‌های دسته‌ای دو-کلاسه (وضعیت تأهل، مالکیت خودرو)
  - رمزگذاری One-Hot برای استان، شغل، و مالکیت خانه
  - رمزگذاری فراوانی (Frequency Encoding) برای شهر (به دلیل تعداد زیاد دسته‌ها)
  - رمزگذاری ترتیبی (Ordinal Encoding) برای ویژگی مهندسی‌شده گروه سنی
- مدل‌سازی
  مقایسه سه مدل رگرسیون لجستیک، درخت تصمیم، و Random Forest با استفاده از اعتبارسنجی متقاطع Stratified K-Fold، با معیار ارزیابی ROC-AUC ( به دلیل عدم توازن کلاس‌ها در متغیر هدف)
- تنظیم هایپرپارامتر
  استفاده از GridSearchCV روی بهترین مدل که رندوم فارست بود
- ارزیابی نهایی 
عدم اکتفا به  roc auc  به دلیل متوازن نبودن داده ها

 نتیجه
پس از تنظیم هایپرپارامتر، مدل Random Forest بهترین عملکرد را داشت که با معیارهای ROC-AUC، H-Measure، و Lift در ۱۰٪ بالای متقاضیان پرریسک  روی داده‌های ارزیابی (validation) سنجیده شد.
